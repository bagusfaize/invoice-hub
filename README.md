## How to run locally

1. Clone this repository and run to your local Terminal
```bash
git clone https://github.com/bagusfaize/invoice-hub.git
```
2. Go to the project folder
```bash
cd .\invoice-hub\
```
3. Install dependencies
```bash
npm install
# or
yarn install
```
4. Run project locally
```bash
npm run dev
# or
yarn dev
```
